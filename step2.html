Step 2: Conduct a Heuristic Evaluation

Screenshot:

Heuristic Evaluation of Craigslist.org

Evaluator: Individual Expert

Website Evaluated: craigslist.org

Date: 21/4/2025

Methodology: Based on Jakob Nielsen’s 10 Usability Heuristics

1. Visibility of System Status

Craigslist provides basic feedback on user actions, such as posting or searching. However, feedback is often minimal or text-only. For example, after submitting a form or posting an ad, users receive confirmation, but there is little visual feedback or progress indicators during longer processes.

Severity: 2 – Minor usability problem

Recommendation: Introduce loading indicators and clearer confirmation messages, especially during posting and account creation.

2. Match Between System and the Real World

Craigslist uses everyday language such as "housing," "jobs," and "for sale," which aligns well with user expectations. However, some categories and subcategories (e.g., “gigs,” “etc/rants”) might confuse new users unfamiliar with site lingo.

Severity: 2 – Minor usability problem

Recommendation: Offer tooltips or brief descriptions for ambiguous categories.

5. Error Prevention

Forms lack input validation and helpful guidance. It’s easy to submit incorrect or incomplete information without immediate feedback.

Severity: 4 – Major usability problem

Recommendation: Introduce real-time validation and clearer input instructions to reduce errors.

6. Recognition Rather Than Recall

Navigation relies heavily on textual lists. While category names are familiar, there is limited use of icons or visual cues to help users quickly recognize options.

Severity: 2 – Minor usability problem

Recommendation: Incorporate icons or visual groupings to reduce cognitive load.

7. Flexibility and Efficiency of Use

Power features exist but are hidden from casual users.

Severity: 3 – Moderate usability problem

Recommendation: Promote power-user features and shortcuts.

8. Aesthetic and Minimalist Design

Minimalism aids usability, but design feels outdated.

Severity: 2 – Minor usability problem

Recommendation: Update visual style while preserving simplicity.

10. Help and Documentation

Craigslist offers a basic help section, but it's not prominent and lacks structure. First-time users might not find the support they need without searching manually.

Severity: 3 – Moderate usability problem

Recommendation: Improve help section layout; integrate contextual help links throughout the site..
